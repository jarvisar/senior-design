How to Use

Initiating a Search

Using one of the four drop-down boxes, enter a valid query to search NASA's exoplanet database. To learn more about each drop-down box, see the input help section below. After entering query, click the search button to fetch data from NASA's database. In order for the search button to be enabled, at least one drop-down box needs to be populated.

Understanding the Input

Hostname refers to the name of the star an exoplanet is orbiting. Discovery method refers to how researchers first discovered an exoplanet. Discovery year refers to the year an exoplanet was first found and the discovery facility refers to which facility first found an exoplanet.

Receiving Results

After receiving returned data from NASA, a table will appear with the formatted results. A download button will also appear which will export the results in CSV format to the local disk. To sort the data, click a column header to toggle between descending and ascending order. Hint: click on an exoplanet's row in the results table to visit the exoplanet's NASA's Eyes page!
